Details

The Ace SD-1 is a single-fire pocket blaster similar to the Alpha Strike Stinger SD-1. It features an integrated dart storage which can also function as an iron sight.

Official description

This is an official description from Hasbro.com. Please do not modify it. (view this product's listing)

Bring along the compact Nerf Elite 2.0 Ace SD-1 blaster on all your missions and battles! Take advantage of this blaster's compact size when you've got to move with stealth. Bring this compact blaster almost anywhere to launch into a stealth games and dart-blasting surprise attacks. It has onboard 1-dart so you can keep a second dart within reach for fast reloads. It comes with 2 Official Nerf Elite foam darts, so you can blast them indoors or outdoors. Get battling in 3 quick steps: Load 1 dart into the front of the barrel, pull the priming handle, and press the trigger to unleash the dart. Eyewear recommended (not included). No batteries required. History

The Ace SD-1 was given a singular release separate of any blaster sets in 2021.

In 2022, it was re-released in Tactical Pack blaster set with a new color scheme.

In 2023, it was reskinned as the Egg Hunt and Jolly Dash.

Color schemes

The Ace SD-1 has been released with the following color schemes:

  • Elite 2.0 (teal, black, and orange)
  • Ace SD-1 Party Pack (orange and black)
  • Tactical Pack (light blue, light green and orange)

Blaster sets

The Ace SD-1 was initially released in the Amazon-exclusive Ace SD-1 Party Pack, which also features an unique orange version of this blaster. In the same year, it was included with Loadout Pack blaster set with no changes to the color scheme. In 2022, it was released in two blaster sets: the Tactical Pack, where it received a new light blue color scheme, and in the Stockpile Pack.

Trivia

  • It is the first Elite 2.0 blaster to come in more than one color scheme.
  • It is tied with the Prospect QS-4 in the number of color variants.
  • It is also the first Elite 2.0 blaster to not feature attachment points of any type.
